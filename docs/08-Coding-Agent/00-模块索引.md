# 08-Coding-Agent

Coding Agent 是能够执行代码任务的 AI Agent。本模块详细介绍 Claude Code 原理、Patch 生成、多文件重构和实战案例。

## 模块目录

- [01-Claude Code 原理](./01-Claude Code 原理.md) - Claude Code 架构与工作原理
- [02-Patch生成机制](./02-Patch生成机制.md) - diff 格式、文件修改策略
- [03-多文件重构](./03-多文件重构.md) - 跨文件修改、依赖处理
- [04-Agent-SWE实战](./04-Agent-SWE实战.md) - SWE-bench、代码修复案例

## 核心概念

```
Coding Agent = Code Understanding + Tool Execution + Result Verification
```

## 关键知识点

| 知识点 | 说明 |
|--------|------|
| Patch Generation | 生成代码修改的 diff |
| Edit Strategy | 单文件/多文件修改策略 |
| Compilation Feedback | 编译错误处理 |
| Test-driven | 测试驱动的开发 |
