# 07-代码理解系统

代码理解系统是 Coding Agent 的核心能力，使其能够"读懂"代码库。本模块详细介绍代码切分策略、代码 Embedding、AST 分析和 Code Graph 构建。

## 模块目录

- [01-代码切分策略](./01-代码切分策略.md) - 按文件、按函数、按语义 Chunk
- [02-代码Embedding](./02-代码Embedding.md) - 代码向量化、语义搜索
- [03-AST代码分析](./03-AST代码分析.md) - 抽象语法树、依赖图
- [04-Code-Graph构建](./04-Code-Graph构建.md) - 代码关系图谱、调用链分析

## 核心概念

### 代码理解系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                    代码理解系统架构                           │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────────┐   │
│  │                    源代码输入                         │   │
│  └─────────────────────────────────────────────────────┘   │
│                           ↓                                  │
│  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐      │
│  │ 代码切分     │  │ 代码向量化   │  │ 语义理解     │      │
│  │ Chunking    │  │ Embedding    │  │ Analysis     │      │
│  ├──────────────┤  ├──────────────┤  ├──────────────┤      │
│  │ • 按文件    │  │ • Token      │  │ • AST 分析   │      │
│  │ • 按函数    │  │ • AST 序列   │  │ • 类型推断   │      │
│  │ • 按语义    │  │ • 图嵌入     │  │ • 依赖关系   │      │
│  └──────────────┘  └──────────────┘  └──────────────┘      │
│                           ↓                                  │
│  ┌─────────────────────────────────────────────────────┐   │
│  │                    索引与检索                        │   │
│  │  • Symbol Index  • Semantic Index  • Graph Index   │   │
│  └─────────────────────────────────────────────────────┘   │
│                           ↓                                  │
│  ┌─────────────────────────────────────────────────────┐   │

│  └─────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────┘
```

### 核心公式

```
Code Understanding = Parsing + Embedding + Indexing + Retrieval
```

## 关键知识点

| 知识点 | 说明 |
|--------|------|
| AST | 抽象语法树，代码的结构化表示 |
| Code Embedding | 将代码转换为语义向量 |
| Symbol Index | 符号索引（函数、类、变量） |
| Data Flow | 数据流分析 |
| Control Flow | 控制流分析 |
| Code Graph | 代码关系图谱 |
