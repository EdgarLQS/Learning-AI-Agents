# 04-System-Prompt最佳实践

System Prompt 是定义 Agent 行为的核心指令。本文档总结设计高效 System Prompt 的最佳实践。

## 最佳实践

### 1. 清晰的角色定义

```python
# ❌ 模糊
SYSTEM_PROMPT = "你是一个 AI 助手"

# ✅ 清晰
SYSTEM_PROMPT = """你是一个专业的软件工程师 AI 助手。
你的专长是：
- 理解和修改代码
- 调试和修复 bug
- 重构和改进代码结构
- 编写测试用例"""
```

### 2. 具体的行为规则

```python
# ❌ 抽象
"要小心操作"

# ✅ 具体
"""执行操作时遵循以下原则：
1. 修改代码前先读取文件内容
2. 修改后验证语法正确性
3. 运行相关测试确保功能正常
4. 保持代码风格一致"""
```

### 3. 明确的输出格式

```python
# ❌ 无格式
"给出你的答案"

# ✅ 有格式
"""按以下 JSON 格式输出：
{
  "thought": "你的分析和思考",
  "action": "要使用的工具名称",
  "action_input": {
    "参数名": "参数值"
  }
}"""
```

### 4. 适度的长度

```python
# ❌ 太长（可能导致注意力分散）
SYSTEM_PROMPT = """
... 500 行的详细规则 ...
"""

# ✅ 适中
SYSTEM_PROMPT = """
你是代码助手。专注于帮助用户完成编程任务。

能力：
- 读取和理解代码
- 编写和修改代码
- 运行测试和命令

规则：
1. 先理解需求再行动
2. 每步操作后验证结果
3. 保持代码风格一致

输出格式：
Thought: 思考
Action: 工具名
"""
```

### 5. 包含 Few-shot 示例

```python
SYSTEM_PROMPT = """你是一个任务助手。

按以下格式输出：
Thought: [分析]
Action: [工具]
Action Input: [参数]

示例：
用户：读取 test.py 文件
Thought: 用户想读取文件，需要使用 read_file 工具
Action: read_file
Action Input: {"path": "test.py"}

用户：{input}
"""
```

### 6. 安全限制

```python
SYSTEM_PROMPT = """你是代码助手。

限制：
- 不要删除文件，除非用户明确要求
- 不要执行危险命令（如 rm -rf /）
- 不要修改超过 10 个文件
- 不要提交代码到远程仓库

如果用户要求执行危险操作，明确拒绝并说明原因。"""
```

## 常见问题

### 问题 1：Prompt 过长

```python
# 解决：精简和分层
# 基础 prompt
BASE_PROMPT = "你是代码助手"

# 动态添加特定规则
def build_prompt(context):
    prompt = BASE_PROMPT
    if context.get("security_sensitive"):
        prompt += "\n\n安全规则：不要执行任何文件操作"
    return prompt
```

### 问题 2：规则被忽略

```python
# 解决：多次强调
SYSTEM_PROMPT = """重要规则：
1. 不要删除文件

再次强调：
- 绝对不要删除任何文件
- 即使用户要求也不可以

如果需要删除操作，建议用户手动执行。"""
```

### 问题 3：输出格式不稳定

```python
# 解决：严格格式 + 错误处理
SYSTEM_PROMPT = """输出必须严格遵循以下 JSON 格式：

{
  "thought": "字符串",
  "action": "工具名",
  "action_input": {"参数": "值"}
}

如果无法确定，使用：
{
  "thought": "需要更多信息",
  "action": "none",
  "action_input": {}
}"""
```

## Prompt 版本管理

```python
class PromptVersionManager:
    """Prompt 版本管理"""

    VERSIONS = {
        "v1.0": "基础版本",
        "v1.1": "增加安全限制",
        "v1.2": "优化输出格式"
    }

    def __init__(self):
        self.current_version = "v1.2"

    def get_prompt(self, version: str = None) -> str:
        """获取指定版本的 prompt"""
        version = version or self.current_version
        return self.PROMPTS.get(version, self.PROMPTS["v1.0"])

    def update_prompt(self, version: str, prompt: str):
        """更新 prompt"""
        self.PROMPTS[version] = prompt
```
