# 04-Prompt-工程

Prompt 工程是充分发挥 LLM 能力的关键技术。本模块详细介绍 Prompt 设计基础、ReAct/CoT 模式、Tool Prompt 设计和 System Prompt 最佳实践。

## 模块目录

- [01-Prompt 设计基础](./01-Prompt 设计基础.md) - Prompt 工程基础与原理
- [02-ReAct与CoT模式](./02-ReAct与CoT模式.md) - 推理模式详解
- [03-Tool-Prompt设计](./03-Tool-Prompt设计.md) - 工具调用提示设计
- [04-System-Prompt最佳实践](./04-System-Prompt最佳实践.md) - 系统提示最佳实践

## 核心概念

### Prompt 工程核心公式

```
LLM Output = f(Prompt, Model Parameters, Context)
```

通过优化 Prompt，可以显著提升模型输出质量，而无需改变模型本身。

## 关键知识点

| 知识点 | 说明 |
|--------|------|
| System Prompt | 定义 Agent 角色和行为规则 |
| ReAct | 推理+行动的协同模式 |
| Chain of Thought | 思维链，引导模型逐步思考 |
| Few-shot Learning | 通过示例引导输出格式 |
| Prompt Injection | 提示注入攻击与防护 |
