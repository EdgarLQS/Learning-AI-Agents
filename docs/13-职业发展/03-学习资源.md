## 学习路径建议

### 0-3 个月：入门

```
Week 1-2: Python 基础
  └─→ 完成 Codecademy Python 课程

Week 3-4: LLM 基础
  └─→ 学习 DeepLearning.AI LLM 课程

Week 5-8: Agent 入门
  └─→ 跟随教程实现简单 Agent

Week 9-12: 项目实战
  └─→ 完成一个完整 Agent 项目
```

### 3-6 个月：进阶

```
Month 4: 深入 LangChain
  └─→ 学习 LangChain 官方文档

Month 5: 代码理解
  └─→ 实现代码索引和检索

Month 6: 性能优化
  └─→ 优化 Agent 成本和速度
```

### 6-12 个月：深入

```
Month 7-8: 系统设计
  └─→ 设计大型 Agent 系统

Month 9-10: 前沿研究
  └─→ 阅读论文，跟进最新进展

Month 11-12: 社区贡献
  └─→ 开源贡献，技术分享
```

## 实用技巧

### 高效学习
1. **边学边做**: 看文档同时动手实践
2. **项目驱动**: 以完成项目为目标学习
3. **问题驱动**: 遇到问题再深入学习
4. **输出倒逼**: 用博客/分享巩固知识

### 信息过滤
1. **关注核心**: 先掌握基础，再拓展
2. **优质优先**: 选择高质量资源
3. **实践验证**: 自己动手验证知识
