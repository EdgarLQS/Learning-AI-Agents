## 核心内容

### 01-技能树
- 基础技术：编程语言、LLM、工具框架
- 核心开发：Agent 系统设计、代码理解、工程能力
- 系统架构：架构设计、稳定性、性能优化
- 算法优化：规划推理、RAG、工具使用

### 02-能力模型
- 初级：基础开发、功能实现
- 中级：独立开发、性能优化
- 高级：架构设计、技术决策
- 资深：技术战略、团队领导

### 03-学习资源
- 书籍：入门到高级推荐
- 课程：免费和付费资源
- 工具：开发框架和监控工具
- 社区：国内外技术社区

## 技能优先级

| 阶段 | 核心技能 |
|------|----------|
| 入门 | Python、LLM API、Git |
| 成长 | Agent 框架、RAG、代码理解 |
| 进阶 | 系统架构、性能优化、团队协作 |
