# 02-向量数据库

向量数据库是实现语义搜索和长期记忆的基础设施。本文档详细介绍主流向量数据库的原理、使用方法和选型指南。

## 向量数据库概述

### 什么是向量数据库

向量数据库专门用于存储和检索高维向量数据，它通过向量Embedding将文本、图像等转换为数学表示，从而实现语义级别的相似度搜索。

```
┌─────────────────────────────────────────────────────────────────┐
│                     向量检索原理                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  原始数据                    向量化                    存储      │
│  ┌──────────┐              ┌──────────┐              ┌────────┐ │
│  │ "Hello   │    ──────→   │ [0.1,    │    ──────→   │ Vector │ │
│  │  World"  │              │  -0.2,   │              │ Store  │ │
│  │          │              │  0.8,..] │              │        │ │
│  └──────────┘              └──────────┘              └────────┘ │
│                                                              ↓    │
│  相似度检索                                                  │    │
│  ┌───────────────────────────────────────────────────────────┐  │
│  │  Query: "Hi there"                                        │  │
│  │     ↓                                                     │  │
│  │  [0.12, -0.19, 0.82, ...]                                │  │
│  │     ↓                                                     │  │
│  │  Top-K 相似向量搜索                                        │  │
│  │     ↓                                                     │  │
│  │  Result: "Hello World" (similarity: 0.92)                │  │
│  └───────────────────────────────────────────────────────────┘  │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 向量数据库 vs 传统数据库

| 特性 | 传统数据库 | 向量数据库 |
|------|-----------|-----------|
| 查询方式 | 精确匹配 | 近似最近邻 (ANN) |
| 检索能力 | 关键词匹配 | 语义相似 |
| 数据类型 | 结构化 | 高维向量 |
| 搜索算法 | B-tree, Hash | HNSW, IVF, LSH |
| 典型场景 | 精确查询 | 语义搜索、推荐系统 |

## 主流向量数据库对比

### 功能对比

| 数据库 | 类型 | 优点 | 缺点 | 适用场景 |

## FAISS 使用详解

### 安装和使用

```bash
pip install faiss-cpu  # 或 faiss-gpu
```

### 基础操作

```python
import faiss
import numpy as np

# 1. 创建向量数据 (假设 embedding 维度为 128)
dimension = 128
n_vectors = 10000

# 随机生成测试数据
vectors = np.random.random((n_vectors, dimension)).astype('float32')

# 2. 创建索引
# 常用索引类型：Flat, IVF, HNSW

# Flat - 精确搜索（暴力计算）
index_flat = faiss.IndexFlatL2(dimension)  # L2 距离
index_flat = faiss.IndexFlatIP(dimension)   # 内积（余弦相似度）

# HNSW - 高效近似搜索（推荐）
index_hnsw = faiss.IndexHNSWFlat(dimension, 32)  # 32 是每层的连接数
index_hnsw.hnsw.efConstruction = 40  # 构建时的搜索宽度
index_hnsw.hnsw.efSearch = 16          # 查询时的搜索宽度

# 3. 添加向量
index_hnsw.add(vectors)

# 4. 搜索
query = np.random.random((1, dimension)).astype('float32')
k = 5  # 返回 top-k 结果

distances, indices = index_hnsw.search(query, k)

print(f"最近邻索引: {indices}")
print(f"距离: {distances}")
```

### 训练索引（IVF 类型）

```python
# IVF 索引 - 聚类 + 精确搜索
# 先训练一个聚类器
nlist = 100  # 聚类数量
quantizer = faiss.IndexFlatL2(dimension)
index_ivf = faiss.IndexIVFFlat(quantizer, dimension, nlist)

# 训练（必须先用数据训练）
index_ivf.train(vectors)

# 添加向量
index_ivf.add(vectors)

# 搜索
index_ivf.nprobe = 10  # 搜索的聚类数
distances, indices = index_ivf.search(query, k)
```

### 持久化

```python
# 保存索引
faiss.write_index(index_hnsw, "index.bin")

# 加载索引
index = faiss.read_index("index.bin")
```

## Chroma 使用详解

### 基础操作

```python
import chromadb
from chromadb.config import Settings

# 1. 初始化客户端
client = chromadb.Client(Settings(
    persist_directory="./chroma_db",
    anonymized_telemetry=False
))

# 2. 创建集合（类似表）
collection = client.create_collection(
    name="documents",
    metadata={"description": "文档向量存储"}
)

# 3. 添加文档
collection.add(
    documents=[
        "Python 是一种高级编程语言",
        "JavaScript 是 Web 前端语言",
        "Rust 是一种系统编程语言",
    ],
    ids=["doc1", "doc2", "doc3"],
    metadatas=[
        {"source": "python-docs", "topic": "language"},
        {"source": "js-docs", "topic": "language"},
        {"source": "rust-docs", "topic": "language"},
    ]
)

# 4. 查询
results = collection.query(
    query_texts=["什么是 Python"],
    n_results=2
)

print(results)
```

### 使用已有 Embedding

```python
from chromadb.api.models import Collection
from sentence_transformers import SentenceTransformer

# 使用自定义 embedding 模型
model = SentenceTransformer('paraphrase-multilingual-MiniLM-L12-v2')

collection = client.create_collection(
    name="custom_embeddings",
    metadata={"hnsw:space": "cosine"}  # 使用余弦相似度
)

# 生成 embedding
texts = ["文本1", "文本2", "文本3"]
embeddings = model.encode(texts).tolist()

# 添加时指定 embedding
collection.add(
    embeddings=embeddings,
    documents=texts,
    ids=["id1", "id2", "id3"]
)

# 查询
query_embedding = model.encode("查询文本").tolist()
results = collection.query(
    query_embeddings=[query_embedding],
    n_results=2
)
```

## 向量检索最佳实践

### 1. 选择合适的索引类型

```python
class VectorIndexSelector:
    """根据场景选择向量索引"""

    @staticmethod
    def select(vectors_count: int, dimension: int, latency_requirement: str) -> str:
        """选择索引类型"""

        if vectors_count < 10000:
            # 小规模数据：Flat 最准确
            return "flat"
        elif latency_requirement == "low":
            # 低延迟要求：HNSW
            return "hnsw"
        else:
            # 大规模数据：IVF
            return "ivf"
```

### 2. 批量操作

```python
def batch_upsert(index, vectors: List[dict], batch_size: int = 100):
```

## 混合搜索策略

### 向量 + 关键词混合

```python
class HybridSearch:
    """混合搜索 - 向量 + 关键词"""

    def __init__(self, vector_index, keyword_index):
        self.vector_index = vector_index
        self.keyword_index = keyword_index

    def search(self, query: str, alpha: float = 0.5, top_k: int = 10):
        """执行混合搜索

        alpha: 向量搜索权重 (1-alpha 为关键词权重)
        """

        # 1. 向量搜索
        vector_results = self.vector_index.query(query, top_k=top_k * 2)

        # 2. 关键词搜索
        keyword_results = self.keyword_index.search(query, top_k=top_k * 2)

        # 3. 结果融合（Reciprocal Rank Fusion）
        rrf_scores = self._reciprocal_rank_fusion(
            vector_results, keyword_results, alpha
        )

        # 4. 返回 top-k
        return sorted(rrf_scores, key=lambda x: x["score"], reverse=True)[:top_k]

    def _reciprocal_rank_fusion(self, vec_results, kw_results, alpha):
        """RRF 融合算法"""
        scores = {}

        # 向量结果评分
        for rank, result in enumerate(vec_results):
            doc_id = result["id"]
            score = (1 - alpha) / (rank + 1)
            scores[doc_id] = scores.get(doc_id, 0) + score

        # 关键词结果评分
        for rank, result in enumerate(kw_results):
            doc_id = result["id"]
            score = alpha / (rank + 1)
            scores[doc_id] = scores.get(doc_id, 0) + score

        return [{"id": k, "score": v} for k, v in scores.items()]
```

## 选型建议

| 场景 | 推荐选择 | 原因 |
|------|----------|------|
| 原型/实验 | Chroma | 简单、Python 原生 |
| 小规模生产 | Qdrant | 性能好、易部署 |
| 大规模生产 | Pinecone | 托管、自动扩展 |
| 高性能需求 | FAISS | 最高性能 |
