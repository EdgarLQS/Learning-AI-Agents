# 06-记忆系统

记忆系统是 Agent 拥有"长期经验"的关键。本模块详细介绍短期记忆、向量数据库、RAG 检索增强和记忆架构设计。

## 模块目录

- [01-短期记忆与上下文](./01-短期记忆与上下文.md) - 会话上下文管理、窗口限制
- [02-向量数据库](./02-向量数据库.md) - FAISS、Chroma、Pinecone 等
- [03-RAG检索增强](./03-RAG检索增强.md) - 知识检索、Chunk策略、重排序
- [04-记忆架构设计](./04-记忆架构设计.md) - 分层记忆、遗忘机制

### 记忆系统的核心公式

```
Agent Memory = Working Context + Conversation History + Retrieved Knowledge
```

## 学习路径

1. **理解上下文管理** - 掌握 LLM 上下文窗口的限制和利用
2. **学习向量数据库** - 了解高效语义检索的技术基础
3. **掌握 RAG** - 学会从知识库中检索相关信息
4. **设计记忆架构** - 构建分层的记忆系统

## 关键知识点

| 知识点 | 说明 |
|--------|------|
| Context Window | LLM 的输入长度限制 |
| Token Budget | 上下文预算的分配策略 |
| Vector Embedding | 文本的语义向量表示 |
| Similarity Search | 基于向量的相似度检索 |
| RAG | 检索增强生成 |
| Memory Hierarchy | 分层记忆架构 |
