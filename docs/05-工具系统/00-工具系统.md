# 05-工具系统

工具系统是 AI Agent 的核心能力之一，它使 LLM 能够与外部世界交互，执行实际操作。本模块详细讲解工具接口设计、调用机制、安全性和内置工具实现。

## 模块目录

- [01-工具接口设计](./01-工具接口设计.md) - OpenAI Tool Calling 格式、函数定义规范
- [02-工具调用机制](./02-工具调用机制.md) - 工具选择、参数校验、执行流程
- [03-工具安全性](./03-工具安全性.md) - 权限控制、危险操作隔离、审计日志
- [04-内置工具实现](./04-内置工具实现.md) - 搜索、读取、编辑、运行命令等

## 核心概念

### 工具系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                      Agent Core                              │
├─────────────────────────────────────────────────────────────┤
│  ┌──────────────┐   ┌──────────────┐   ┌──────────────┐    │
│  │ Tool Selector│→  │ Param Parser │→  │ Tool Executor│    │
│  └──────────────┘   └──────────────┘   └──────────────┘    │
├─────────────────────────────────────────────────────────────┤
│                    Tool Registry                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  tool_definition_1  │  tool_definition_2  │  ...    │   │
│  └─────────────────────────────────────────────────────┘   │
├─────────────────────────────────────────────────────────────┤
│                    Tool Implementations                      │
│  ┌─────────┐  ┌─────────┐  ┌─────────┐  ┌─────────┐        │
│  │ 搜索工具 │  │读取工具 │  │编辑工具 │  │执行工具 │        │
│  └─────────┘  └─────────┘  └─────────┘  └─────────┘        │
└─────────────────────────────────────────────────────────────┘
```

### 工具系统的核心公式

```
Tool System = Tool Definition + Tool Calling + Tool Execution + Result Parsing
```

## 学习路径

1. **理解工具接口格式** - 掌握 OpenAI Tool Calling 规范
2. **学习工具调用流程** - 了解从选择到执行的全过程
3. **关注安全性** - 学会安全地暴露工具给 Agent
4. **实践内置工具** - 掌握常见工具的实现方式

## 关键知识点

| 知识点 | 说明 |
|--------|------|
| Tool Calling | LLM 生成调用工具的意图和参数 |
| Function Schema | 工具函数的 JSON Schema 定义 |
| Tool Registry | 工具注册表，管理可用工具 |
| Sandbox Execution | 沙箱执行，确保工具安全 |
| Result Parsing | 解析工具返回结果并反馈给 LLM |
