# Agent 框架对比

> 主流 Agent 框架的对比与特点

## 框架概览

| 框架 | 类型 | 特点 | 文档链接 |
|------|------|------|----------|
| [AutoGPT](./01-AutoGPT%20详解.md) | 通用 Agent | 自主任务执行、递归优化 | [AutoGPT](https://github.com/Significant-Gravitas/AutoGPT) |
| [LangChain/LangGraph](./02-LangGraph.md) | 框架 | 流程编排、状态管理 | [LangChain](https://www.langchain.com/) · [LangGraph](https://langchain-ai.github.io/langgraph/) |
| [CrewAI](./03-CrewAI.md) | 多 Agent | 角色协作、任务分派 | [CrewAI](https://www.crewai.com/) |

## 选型建议

- **简单任务**: AutoGPT
- **复杂流程**: LangGraph
- **多角色协作**: CrewAI
